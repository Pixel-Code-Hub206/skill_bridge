# SkillBridge

SkillBridge is a Student Skills Management System designed to help educational institutions identify and match students based on skills and proficiency levels.

Students create and manage skill profiles, while teachers define project requirements, search suitable candidates, and send invitations. The system ensures transparency, efficient shortlisting, and fair opportunity distribution.

## Features
- Student registration and profile management
- Skill addition with proficiency levels
- Teacher project creation and skill-based search
- Student ranking and shortlisting
- Invitation and status tracking
- Secure authentication and role-based access

## Tech Stack
**Frontend:** HTML, CSS, JavaScript (React optional, Tailwind CSS optional)  
**Backend:** Java with Spring Boot  
**Database:** PostgreSQL (containerized using Docker for isolated local development)
**Tools:** Git, GitHub, Postman
